This is a simple case I modified from (https://makerworld.com/en/models/2108947-esp32-c3-pn532-nfc-reader-case-usb-c#profileId-2552448). 

The changes I made are adding Text to the model so we can have each Toolhead labeled (T0, T!, T2 etc.) 
I also modified the area where the esp32 goes. This model is designed for the waveshare esp32-s3-zero (get the one without the pins and directly solder wires to it)
I also added a target where you scan your tag. I suggest you make this Red.

For full effect print the case itself in a clear material (PLA, Petg, PC) so you can get the full effect of the LED.
